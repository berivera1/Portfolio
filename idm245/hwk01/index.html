CrossCode

Play Game

CrossCode is a 2D top-down RPG game made with Javascript and Impact JS. It utilizes the WASD control scheme and the mouse.

The Good

I enjoyed this game because of its smooth controls, a unique art style, a complete soundtrack, a gripping story, and it feels fun to play. Everything about the game feels polished, and it all comes together to make a very fun, enjoyable experience.

The Bad

The only aspect of the game that could be improved on is objective communication to the player. The initial tutorial is very straight-forward, but in the main area of the demo (the ship), it becomes easy as a player to get lost, and not much is done in terms of UI to guide the player in the right direction. Though exploring the ship is fun, I found myself getting lost and annoyed during my initial playthrough because there was no obvious indicator of where to go.

Knightin'

Play Game

Knightin' is a 2D top-down Dungeon Crawler game made with Construct 2, an engine running off of HTML5 and Javascript.

The Good

I enjoyed this game because of its vibrant color palette, smooth controls, and a good soundtrack. It has an easy to learn tutorial that integrates teaching the player main game mechanics with the game environment. This makes the entire process of learning the game feel natural, even on first playthrough.

The Bad

The game tries to create a unique mechanic by intergrating QR codes into its puzzles. Because this requires a phone, though, this ends up making these puzzles feel more like a chore than a fun mechanic. The game also suffers from a lack of a large range of enemies, and after a while the gameplay becomes repetitive.
